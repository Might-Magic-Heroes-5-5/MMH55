1. Download debug.ps1 and directory_map.txt and place them in the game folder.
2. Then as you are in the folder select  File ->  Open Windows Powershell -> Open windows powershell as admininistrator
![image](https://github.com/user-attachments/assets/ec0c92a7-262d-4ceb-ab22-1095c6877bfc)
3. Type command and hit enter
```powershell
    Set-ExecutionPolicy unrestricted
```
4. Agree with the question asked by typing Y and hitting enter.
```powershell
    Y
``` 
5. Type command and hit enter
```powershell
    .\debug.ps1
```
6. Script will create M55_debug.txt file in root folder of the game. Send it for debug.
